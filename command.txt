■変更済みファイルを取得する
 git diff --stat --name-only

■HEADのリビジョンを取得する
 git refv-parse HEAD

■現在のブランチ名を取得する
 git rev-parse --abbrev-ref HEAD


■コマンド
gcc -DGIT_BRANCH=`git rev-parse --abbrev-ref HEAD` -DGIT_REVISION=`git rev-parse HEAD` -DGIT_MODIFICATIONS=\""`git diff --stat --name-only | awk -F\n -v ORS=','  '{print}' | sed -e 's/,$//' `"\" sample_main.c
